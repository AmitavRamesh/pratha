# Sustain-ify

Sustain-ify is a production-grade, eco-conscious AI application.

## Structure
- `/frontend` - Next.js (App Router), TypeScript, Tailwind CSS, shadcn/ui.
- `/backend` - Python (FastAPI), Async architecture.

## Deployment
- Frontend: Vercel
- Backend: Render

## Environment Setup
See `/backend` and `/frontend` for detailed setup steps.
