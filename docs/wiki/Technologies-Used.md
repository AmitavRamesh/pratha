# Technologies Used

## Programming Languages
*   **TypeScript / JavaScript**: Powering the Next.js frontend with strong type safety.
*   **Python**: Used for the FastAPI backend and AI integration scripts.

## Frameworks / Libraries
*   **Next.js 15**: React framework for building the user interface.
*   **FastAPI**: High-performance Python framework for building APIs.
*   **GenAI SDK**: Google's official SDK for interacting with Gemini models.
*   **Pydantic**: Data validation and settings management using Python type annotations.
*   **Uvicorn**: ASGI server for running the FastAPI application.

## Tools
*   **Git / GitHub**: For version control and collaborative development.
*   **Google Gemini AI**: Specifically the `gemini-3-flash-preview` model for multimodal analysis.
*   **Open Food Facts API**: The primary source for global product data.
*   **NPM / Node.js**: Package management for the frontend.
*   **Venv**: For Python dependency isolation.

## Hardware
*   **Testing Devices**: Modern smartphones and laptops with high-resolution cameras for testing scanning capabilities.
