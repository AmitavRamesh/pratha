# Challenges and Limitations

## Challenges Faced During Development
*   **Gemini Rate Limits**: Managing API quotas during development cycles.
*   **OCR Noise**: Handling handwritten or partially obscured text on crumpled product packaging.
*   **Video Processing**: Implementing a robust wait-mechanism for the Gemini Files API while processing video files.
*   **Frontend Camera Access**: Ensuring cross-browser compatibility for direct camera streams in Next.js.

## Limitations of the System
*   **Connectivity**: Requires a stable internet connection to communicate with Gemini and Open Food Facts.
*   **Ambient Lighting**: Scan quality decreases significantly in low-light environments.
*   **Language Support**: Primarily optimized for English labels; multilingual label support is currently limited.
