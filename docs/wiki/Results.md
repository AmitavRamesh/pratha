## Performance Metrics
*   **API Response Time**: ~500ms for barcode lookup via Open Food Facts.
*   **AI Analysis Time**: ~3-5 seconds for image/video processing via Gemini.
*   **Accuracy**: High extraction accuracy (>90%) for clearly legible packaging labels.

## Observations
*   The system successfully identifies regional snacks that are missing from global databases using AI.
*   Videos provide better context for analysis than single blurry photos.
*   Eco-scores are highly effective in alerting users about carbon-heavy products like tropical oils or air-shipped goods.
